# Poke Three (炸金花)

一个在线多人炸金花扑克游戏，支持实时对战。

## 技术栈

- **后端**: Java 21 + Spring Boot 3 + MyBatis-Plus + WebSocket + SQLite
- **前端**: Vite + Vanilla JS + TailwindCSS

## 项目结构

```
poke_three/
├── backend/        # Spring Boot 后端服务
├── frontend/       # Vite 前端项目
└── README.md
```

## 快速开始

### 后端

```bash
cd backend
mvn spring-boot:run
```

### 前端

```bash
cd frontend
npm install
npm run dev
```
